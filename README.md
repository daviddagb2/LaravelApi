
# Ejemplo de Api Laravel 9

Este proyecto ha sido desarrollado con la finalidad de mostrar como se puede construir una API sencilla utilizando PHP Laravel 9 y como pueden crearse los metodos GET, POST, PUT y DELETE.


## Authors

- [David Gonzalez Blanchard](https://github.com/daviddagb2)


## Requerimientos para probar este proyecto

 - [PHP 8.1](https://www.php.net/releases/8.1/en.php)
 - [MySQL](https://www.mysql.com/)
 - [Composer](https://getcomposer.org/)
 - [Laravel 9.X](https://laravel.com/docs/9.x/installation)
 - [Visual Studio Code](https://code.visualstudio.com/)
